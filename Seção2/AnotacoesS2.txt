Anotações:

- Console.log() -> é a saida do JS.

- let == var (mas tem diferenças).

- Variaveis podem ser mudados.
- nome = "Gustavo" -> isso cria uma variavel global, pois não tem var ou let antes.
- Constantes não podem mudar -> ja tenho que colocar um valor na constante.

- Usar o template strings para fazer saida.

- + -> adição e concatenação (com string).

- parseInt('num') -> converte para number.
- parseFloat().
- Number() -> qualquer um dos dois.

- alert(); -> alerta na tela.
- confirm(); -> caixa com "sim" e "não".
- prompt(); -> caixa para digitar dentro.

- ixdexof() e lastIndexOf() -> mostra o indice que tal string ta dentro de outra.
- replace(--,--) -> troca uma string por outra.
- length -> tamanho da string.
- slice(--,--) -> fatia.
