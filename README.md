# 📚 Curso JS e TS

Repositório com anotações, exercícios e projetos desenvolvidos durante um curso de **JavaScript e TypeScript** realizado na Udemy.

Este repositório acompanha minha evolução ao longo das seções do curso, reunindo exemplos práticos e aplicações dos conceitos estudados.

---

## 🚀 Sobre o repositório

Aqui você encontrará:

- 📝 Anotações das aulas  
- 💻 Exercícios práticos  
- 🧩 Exemplos de aplicação dos conceitos  
- 🔄 Manipulação de DOM  
- 📦 Uso de localStorage  
- ⚙️ Conceitos de programação orientada a objetos  
- 🧠 Introdução e aprofundamento em TypeScript  

Os conteúdos estão organizados por seções, seguindo a estrutura do curso.

---

## 🛠️ Tecnologias utilizadas

- **JavaScript**
- **TypeScript**
- **HTML**
- **CSS**

---

## 🎯 Objetivo

Este repositório tem como objetivo:

- Consolidar fundamentos de JavaScript  
- Aprender tipagem estática com TypeScript  
- Praticar manipulação de dados e lógica  
- Desenvolver aplicações web modernas  
- Registrar minha evolução durante o curso
