Anotações:

- Operadores de comparação ão iguais ao da matemática.
- Usar os estritos porque eles olham para o VALOR e o TIPO.

- Operados lógicos seguem a mesma lógica de sempre.

- Curto-Circuito (short-circuit evaluation)
  - Analisar da esquerda para direita. Quando encontra algo que define o final, retorna o valor e não TRUE ou FALSE.
  && -> Retorna o primeiro false, null, undefined, 0, "", NaN.
     -> Se nenhum for falso, retorna o ultimo.

  || -> Retorna o primeiro verdadeiro.
     -> Se nenhum for verdadeiro, retorna o ultimo.

- if, else if e else -> igual sempre.

- Operados ternários
   - condição ? caso for verdade : caso for falso

- Switch/Case -> igual sempre.

- Let  tem escopo de bloco {... bloco}.
- Var só tem escopo de função.

- Tem como ter array dentro de array. Ex: array[i][j].
- Usa a virgula para pular o que não quer do array (metodo pior).

- Se achar algo que esta dentro do objeto, pelo mesmo nome, é bem mais facil.

- for in -> for (let i in NOME) -> retorna o indice
- for of -> for (let i of NOME) 0> retorna o valor -> não funciona com objetos
